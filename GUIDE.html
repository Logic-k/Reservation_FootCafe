▶️ 이용 시작하기 (손님 입장 시)

🧑 1인 이용

  1. 좌석현황 탭에서 초록색 좌석을 탭
  2. "이용 시작" 모달이 열림
  3. [1인 이용 시작] 버튼 탭
  4. 좌석이 주황색으로 바뀌고, 40분 타이머 시작!

👨‍👩‍👧‍👦 그룹 이용 (2인 이상 - 커플/가족/여행객)

  1. 초록색 좌석 하나를 탭
  2. [그룹 이용 시작 (2인 이상)] 버튼 탭
  3. 그룹명 입력 (예: "커플A", "가족김씨", "여행객3인")
  4. 함께 이용할 추가 좌석을 탭하여 선택 (선택된 좌석은 보라색)
  5. [그룹 시작 (N인)] 버튼 탭
  6. 선택한 모든 좌석이 동시에 "이용 중"으로 변경됨!

⏹️ 이용 종료하기 (손님 퇴장 시)

  1. 주황색(이용 중) 좌석을 탭
  2. 이용 정보 확인 (시작 시간, 종료 예정, 남은/초과 시간)
  3. [이용 종료] 버튼 탭
  4. "정말 종료하시겠습니까?" → [종료] 탭

그룹 종료 시 선택지

버튼동작
[이것만]이 좌석만 종료 (나머지 그룹은 유지)
[전체 종료]같은 그룹 좌석 모두 동시 종료
[취소]아무것도 하지 않음

📝 예약 등록하기 (좌석이 꽉 찼을 때)

  1. 예약등록 탭으로 이동
  2. 필수 정보 입력:
    • 예약 시간 — 언제 이용할 예정인지 (미래 시간만 가능)
    • 예약자명 — 손님 이름/닉네임 (1~20자)
    • 결제방식 — 현금 / 계좌이체 / 카드
    • 좌석 선택 — 원하는 좌석을 탭 (파란색으로 선택됨)
  3. 2인 이상 → 여러 좌석 탭 시 "그룹명" 입력란 자동 표시
  4. [예약 등록] 버튼 탭
⚠️ 시간이 겹치는 좌석은 자동으로 차단됩니다. 겹침 발생 시 노란색 경고 메시지가 표시됩니다.

추천 좌석 (예약등록 탭 하단)

  • "지금 이용 가능" → 현재 비어있는 좌석
  • "HH:MM 이후 이용 가능" → 해당 시간 이후에 비는 좌석

❓ 자주 묻는 질문 (FAQ)

Q: 이용 시간은 왜 40분인가요?
A: 족욕 30분 + 전후 여유시간 10분 = 총 40분으로 설정되어 있습니다.
Q: 시간이 초과되면 자동으로 종료되나요?
A: 아닙니다. 직원이 직접 [이용 종료] 버튼을 눌러야 합니다. 초과 시간은 화면에 표시됩니다.
Q: 같은 좌석에 예약이 겹칠 수 있나요?
A: 불가능합니다. 시스템이 자동으로 시간 충돌을 감지하고 차단합니다.
Q: 앱 데이터가 날아가면 어떡하나요?
A: Firebase 클라우드에 실시간 저장되므로 기기를 바꿔도 데이터가 유지됩니다.
Q: 그룹 이용 중 한 명만 먼저 나가면?
A: 해당 좌석만 개별 종료할 수 있습니다. [이것만] 버튼을 사용하세요.

🛠️ 문제 해결

증상해결 방법
화면이 안 바뀜새로고침 (당겨서 새로고침 또는 F5)
다른 기기와 동기화 안됨인터넷(와이파이) 연결 확인
좌석이 안 눌림초록/주황/파란색 좌석만 탭 가능
예약 등록 안됨모든 필수 항목 입력 확인, 미래 시간인지 확인
