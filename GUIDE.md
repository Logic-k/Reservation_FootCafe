# 족욕카페 좌석 예약 관리 - 사용 가이드

---

## 📱 앱 접속 방법

브라우저(크롬, 사파리 등)에서 배포된 URL을 입력하면 바로 사용할 수 있습니다.
PC, 태블릿, 스마트폰 모두 동일하게 사용 가능합니다.

> 💡 **팁**: 스마트폰에서 "홈 화면에 추가"를 하면 앱처럼 바로 열 수 있습니다.

---

## 🏠 화면 구성

앱은 상단에 **4개의 탭**으로 구성되어 있습니다:

| 탭 | 기능 |
|----|------|
| **좌석현황** | 12개 좌석의 현재 상태를 한눈에 확인 |
| **예약등록** | 임시예약을 새로 등록 |
| **예약목록** | 등록된 예약을 확인/관리 |
| **이용로그** | 이용 시작/종료, 예약 기록 확인 |

---

## 🪑 좌석현황 탭

### 좌석 배치

```
[입구]        [창가]
입구4    1번 2번 3번 4번 5번 6번 7번 8번
입구3
입구2
입구1
```

### 좌석 색상 의미

| 색상 | 상태 | 의미 |
|------|------|------|
| 🟩 초록색 | 비어있음 | 새 손님 이용 가능 |
| 🟧 주황색 | 이용 중 | 손님이 사용 중 (남은 시간 표시) |
| 🟦 파란색 | 예약됨 | 예약된 좌석 (예약자명 표시) |
| 🔴 빨간 테두리 깜빡임 | 종료 임박 | 5분 이내에 시간 종료 예정 |

### 좌석에 보라색 태그가 보이면?
→ **그룹**(커플/가족/일행)으로 함께 이용 중인 좌석입니다.

---

## ▶️ 이용 시작하기 (손님 입장 시)

### 1인 이용

1. **좌석현황** 탭에서 초록색(비어있음) 좌석을 탭
2. "이용 시작" 모달이 열림
3. **[1인 이용 시작]** 버튼을 탭
4. 좌석이 주황색으로 바뀌고, 40분 타이머 시작

### 그룹 이용 (2인 이상 - 커플/가족)

1. 초록색(비어있음) 좌석 하나를 탭
2. **[그룹 이용 시작 (2인 이상)]** 버튼을 탭
3. **그룹명** 입력 (예: "커플A", "가족김씨", "여행객3인")
4. 함께 이용할 **추가 좌석**을 선택 (보라색으로 변함)
5. **[그룹 시작 (N인)]** 버튼을 탭
6. 선택한 모든 좌석이 동시에 이용 중으로 변경됨

---

## ⏹️ 이용 종료하기 (손님 퇴장 시)

1. 주황색(이용 중) 좌석을 탭
2. 이용 정보 확인 (시작 시간, 종료 예정, 남은/초과 시간)
3. **[이용 종료]** 버튼 탭
4. "정말 종료하시겠습니까?" 확인 → **[종료]** 탭

### 그룹 종료

그룹 좌석을 탭하면 3가지 선택지:
- **[이것만]**: 이 좌석만 종료 (나머지 그룹은 유지)
- **[전체 종료]**: 같은 그룹 좌석 모두 동시 종료
- **[취소]**: 아무것도 하지 않음

---

## 📝 예약 등록하기 (좌석이 꽉 찼을 때)

1. **예약등록** 탭으로 이동
2. 필수 정보 입력:
   - **예약 시간**: 언제 이용할 예정인지 (미래 시간만 가능)
   - **예약자명**: 손님 이름/닉네임 (1~20자)
   - **결제방식**: 현금 / 계좌이체 / 카드 중 선택
   - **좌석 선택**: 원하는 좌석을 탭 (파란색으로 선택됨)
3. **2인 이상 예약 시**: 여러 좌석을 탭하면 "그룹명" 입력란이 나타남
4. **[예약 등록]** 버튼 탭

> ⚠️ 시간이 겹치는 좌석은 자동으로 차단됩니다. 겹침 발생 시 경고 메시지가 표시됩니다.

### 추천 좌석

예약등록 탭 하단에 **추천 좌석** 목록이 표시됩니다:
- "지금 이용 가능" → 현재 비어있는 좌석
- "HH:MM 이후 이용 가능" → 해당 시간 이후에 비는 좌석

추천 좌석을 탭하면 자동으로 선택됩니다.

---

## 📋 예약 목록 관리

**예약목록** 탭에서 등록된 모든 예약을 확인할 수 있습니다.

각 예약 항목에는 2개의 버튼이 있습니다:

### [전환] 버튼 (초록색)
예약된 손님이 도착했을 때 사용합니다.
- 예약 → 이용 시작으로 즉시 전환
- 좌석 상태가 "예약됨" → "이용 중"으로 바뀜
- ⚠️ 좌석이 아직 사용 중이면 전환 불가 (에러 메시지 표시)

### [취소] 버튼 (빨간색)
예약을 취소합니다.
- "이 예약을 취소하시겠습니까?" 확인 후 삭제
- 좌석이 다시 "비어있음"으로 변경됨

---

## 📊 이용 로그

**이용로그** 탭에서 모든 활동 기록을 확인합니다.

| 배지 색상 | 기록 유형 |
|-----------|-----------|
| 🟢 초록 | 이용 시작 |
| ⚪ 회색 | 이용 완료 |
| 🔵 파랑 | 예약 등록 |
| 🔴 빨강 | 예약 취소 |
| 🟣 보라 | 예약→이용 전환 |

### 로그 삭제
- 개별 삭제: 각 로그 항목에 마우스를 올리면 ✕ 버튼 표시
- 전체 삭제: 상단 "전체 삭제" 버튼 → 확인 후 일괄 삭제

---

## ⏰ 자동 알림

### 종료 임박 알림 (5분 전)
- 좌석 카드 테두리가 **빨간색으로 깜빡임**
- 손님에게 시간 거의 끝남을 안내해주세요

### 시간 초과 알림
- 화면 상단에 **빨간색 알림 배너** 표시
- "⚠️ OO번 좌석이 N분 초과되었습니다"
- ✕ 버튼으로 배너를 닫을 수 있음

## 🔄 기기 간 동기화

이 앱은 **Firebase**로 실시간 동기화됩니다:
- 카운터 PC에서 이용 등록 → 사장님 폰에서 즉시 반영
- 모바일에서 예약 등록 → 다른 기기에서도 바로 보임
- 인터넷 연결이 끊어져도 로컬에 저장되어 사용 가능

## ❓ 자주 묻는 질문

### Q: 이용 시간은 왜 40분인가요?
A: 족욕 30분 + 전후 여유시간 10분 = 총 40분으로 설정되어 있습니다.

### Q: 시간이 초과되면 자동으로 종료되나요?
A: 아닙니다. 직원이 직접 [이용 종료] 버튼을 눌러야 합니다. 초과 시간이 표시되어 안내에 활용할 수 있습니다.

### Q: 같은 좌석에 예약이 겹칠 수 있나요?
A: 불가능합니다. 시스템이 자동으로 시간 충돌을 감지하고 차단합니다.

### Q: 앱 데이터가 날아가면 어떡하나요?
A: Firebase 클라우드에 실시간 저장되므로 기기를 바꿔도 데이터가 유지됩니다.

### Q: 그룹 이용 중 한 명만 먼저 나가면?
A: 해당 좌석만 개별 종료할 수 있습니다. [이것만] 버튼을 사용하세요.

## 🛠️ 문제 해결

| 증상 | 해결 방법 |
|------|-----------|
| 화면이 안 바뀜 | 새로고침 (당겨서 새로고침 또는 F5) |
| 다른 기기와 동기화 안됨 | 인터넷 연결 확인 |
| 좌석이 안 눌림 | 초록/주황/파란색 좌석만 탭 가능 |
| 예약 등록 안됨 | 모든 필수 항목 입력 확인, 미래 시간인지 확인 |
